📚 Git & GitHub Notes

What is Git?

Git is a Version Control System (VCS) used to track changes in source code during software development. It helps developers save project history, manage different versions, and collaborate with team members efficiently.

  • ✔ Tracks code changes
  • ✔ Saves project history
  • ✔ Restores previous versions
  • ✔ Supports teamwork
  • ✔ Works offline

What is GitHub?

GitHub is a cloud-based platform that allows developers to store, manage, and share Git repositories online. It provides collaboration features, project hosting, and backup for software projects.

  • ✔ Stores repositories online
  • ✔ Backup your projects
  • ✔ Share code with others
  • ✔ Collaborate with teams
  • ✔ Access projects from anywhere

Difference Between Git & GitHub

  • Git is a Version Control System.
  • GitHub is a cloud platform for Git repositories.
  • Git works on your local computer.
  • GitHub stores your project online.
  • Git can work without the internet.
  • GitHub requires the internet to upload and share projects.
